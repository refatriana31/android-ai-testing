Rekomendasi

1. Siapkan Quiz & Konten Berskor (~12 skenario)
Buat minimal 1 essay quiz + 1 MCQ dengan ambang kelulusan → membuka LIB-M-012/014/015, FLOW-M-006/007.
2. Sediakan Akun Manager (7 skenario)
Credential manager-role di permata-revamp-stg → membuka TEAM-M-001 sampai TEAM-M-008.
3. Tambah Video/Audio/PDF (~6 skenario)
Katalog saat ini hanya SCORM. Minimal 1 video + 1 audio + 1 PDF → membuka CON-M-001/002/007.
4. Event Publik Baru (~7 skenario)
Semua event sudah terdaftar. 1 event publik baru → membuka EVT-M-003/004 dan turunannya.
5. Akun Disposable (~5 skenario)
Tes yang mengubah state (Like, bookmark, profile edit) bisa jalan di akun sekali pakai.
